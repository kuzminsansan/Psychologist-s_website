# Отдельная версия сайта

Эта папка создана как отдельная ветка разработки в формате отдельной директории.

Рекомендуемая структура для следующего шага:
- `react-version/src/`
- `react-version/components/ui/`
- `react-version/index.css`

Сюда можно безопасно перенести React + TypeScript + Tailwind + shadcn реализацию, не затрагивая текущую рабочую версию лендинга в корне проекта.
